🔧 Install from source

Requires Node.js 22+ and git.

$ git clone https://github.com/aegisinfo/aegiscode.git $ cd aegiscode $ npm install $ npm run build $ ./aegis

Or install globally: npm install -g aegiscode

🚀 Quick start

$ aegis # First launch will prompt for API key(s). # Add keys to .env or use /model to switch: $ /model claude $ /model deepseek $ /model groq $ /council "review this diff"
